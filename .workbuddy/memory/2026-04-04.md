## 库存接口传参修正（重要！）

`listPrSingleOrderRoom` 接口正确传参格式：
- **日期格式必须用 `YYYY-MM-DD`（带横线）**，不是 `YYYYMMDD`
- `hotelCodes` 必须是数组格式，不是单个字符串 `hotelCode`
- 必传固定参数：`hotelGroupCode: "SONGTSAM"`, `otaChannel: "CRS"`, `maxMemberLevel: "002"`, `unitCode: "SONGTSAM"`

正确示例：
```json
{
  "hotelGroupCode": "SONGTSAM",
  "otaChannel": "CRS",
  "maxMemberLevel": "002",
  "unitCode": "SONGTSAM",
  "hotelCodes": ["STLKBSC"],
  "beginDate": "2026-06-20",
  "endDate": "2026-06-21"
}
```

返回字段说明：
- `rmtype` / `rmtypeDesc`：房型代码/描述
- `detailDtos[].saleNum`：可售总量
- `detailDtos[].blockNum`：已封房数量
- `detailDtos[].costPrice`：成本价
- `detailDtos[].rsvDate`：对应日期
- 剩余房间 = saleNum - blockNum
