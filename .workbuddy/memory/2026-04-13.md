## 库存API修复（2026-04-13 下午）

**问题**：查"7月1日巴松措有没有房"返回"没有"，实际有17间可售

**根因**：
1. 库存API（gds.songtsam.com/product-room）需要请求体中带 `userCode` 字段
   - 正确值：从 JWT token 解码出的 `userCode='HL'`（不是登录手机号）
   - 后端之前没传，导致 HTTP 500 "userCode is Missing"
2. API 返回的 `sta=null`，但过滤条件 `sta !== 'I'` 把所有数据都过滤掉了

**修复**：
- `recommend_api.py`: 新增 `_get_user_code_from_token()` 从 JWT 解码 userCode，加到请求体
- 过滤逻辑：改为 `(sta in (None, '', 'I'))`
- `产品推荐助手-AI版.html`: 新增 `getUserCodeFromToken()`，queryInventory 加 userCode
- `产品推荐助手.html`: 同上sta过滤修复

**Git**: `fix: 库存API加userCode参数+修复sta=null被过滤`
