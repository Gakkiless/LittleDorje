# 2026-04-03 工作日志

## 计价工具 v2.1 更新

### 利润率API修复
- 修改 `initProfitRates()` 根据 itemCode 精确匹配季节：
  - `GROUP_SHARING_TAX_RATE` → 集团分摊与息税率（API返回25.6%）
  - `BASIC_COST_RATE` → 集团基础利润率（API返回19.63%）
  - `HOLIDAY_PROFIT_RATE` → 节日
  - `PEAK_PROFIT_RATE` → 旺季
  - `FLAT_PROFIT_RATE` → 平季
- **移除淡季**：API 只返回三个季节，淡季完全移除
- 公式显示更新为：`利润率 = 集团分摊与息税率(25.6%) + 集团基础利润率(19.63%) − 季节成本率`

### 新增：酒店日历API集成
- 接口：`POST https://api.songtsam.com/quotation_center/bks/hotelCalendar/pageQuery`
- 参数：`{hotelGroupCode: "SONGTSAM", pageNum: 1, pageSize: 200}`
- 返回：酒店日历列表，每个日历包含：
  - `hotelCode`: 酒店代码
  - `startDate`/`endDate`: 季节生效日期范围
  - `name`: 日历名称（如"拉萨区-旺季【桃花节4/1-4/20】"）
  - `profitType`: 关联利率项（节日利润率/旺季利润率/平季利润率）

### 简化季节选择
- 移除日期输入和自动匹配功能
- 只保留手动选择季节：节日 | 旺季 | 平季
