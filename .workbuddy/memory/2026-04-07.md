## 向量库建设完成

### 已创建文件
- `sync_products_to_vectorstore.py` - 产品数据同步脚本（API → ChromaDB）
- `recommend_products.py` - 产品推荐脚本（语义搜索 + 推荐算法）
- `chroma_db/` - 向量库存储目录

### 技术栈
- 向量库: ChromaDB（本地持久化存储）
- Embedding: Ollama + `nomic-embed-text`（本地模型）
- Python: 3.14（虚拟环境 `.venv`）

### 数据规模
- 577个产品全部入库
- 每条数据含：产品名称/环线/天数/类型/集合散团地/标签/完整行程描述

### 产品详情接口（已确认）
- 基础信息: `GET /product-journey/bks/travelproduct/getTravelProductType?travelType=xxx&unitCode=SONGTSAM`
- 行程详情: `GET /product-journey/bks/itinerary/getTravelProductitinerary?travelType=xxx&unitCode=SONGTSAM`

### 推荐算法
- 语义搜索：基于 Ollama embedding 相似度匹配
- 规则评分：天数匹配/类型匹配/季节主题匹配/目的地匹配
- 综合排序：语义相关度 × 规则评分

### 使用方法
```bash
# 同步产品到向量库（数据更新时运行）
./.venv/bin/python3 sync_products_to_vectorstore.py

# 推荐+团期+价格（主力脚本）
./.venv/bin/python3 recommend_with_groups.py "去林芝看桃花"

# 纯推荐（不用API）
./.venv/bin/python3 recommend_products.py "深度户外"
```
